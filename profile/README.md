## ISIS Experiment Controls

The experiment controls team is responsible for the control system software used by the [ISIS Neutron and Muon Source](https://www.isis.stfc.ac.uk/)'s
instruments, from detector data acquisition and beamline equipment control, through to data-file generation, experiment scripting, and high-level
user interfaces.

Quick links:
- [Web Dashboard](https://isiscomputinggroup.github.io/WebDashboard/instruments) - monitor the status of your experiment
- [IBEX User Manual](https://isiscomputinggroup.github.io/ibex_user_manual/) - including details of how to contact the team
- [IBEX Developer Manual](https://isiscomputinggroup.github.io/ibex_developers_manual/) - detailed information for developers
